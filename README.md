# IDS Dashboard Creation with Suricata in Splunk

## Overview
This project demonstrates the setup and configuration of an Intrusion Detection System (IDS) using Suricata and visualizing the data in Splunk through a comprehensive dashboard.

## Objectives
- Set up Splunk and Suricata for network monitoring.
- Ingest Suricata logs into Splunk.
- Create dashboards to visualize IDS data.
- Configure alerts for real-time monitoring.

## Prerequisites
- Splunk Enterprise or Splunk Cloud
- Suricata IDS
- Sufficient disk space and memory
- Administrative access to network and systems being monitored
